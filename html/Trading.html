Trading Platform:

J-LAB


I began studying the markets using TradeStation’s EasyLanguage 9. I quickly realized the need for a more feature rich platform and a lower level language. I then migrated to NinjaTrader’s NinjaScript 8, which allowed me to build much more powerful strategies and indicators. However, in order to truly study the markets the way I wanted to, I realized it just can’t be done using someone else’s software. So, I built my own.

J-LAB allows me full access to the power of C#, OOP, visualizations, and the powerful debugging tools that Microsoft Visual Studio offers. In this way, I can develop naturally, fix my own bugs immediately, and add useful features at will. This opens up the possibility to do Machine Learning studies, proper Statistical Analysis, connect to any API, and even leverage other languages.
